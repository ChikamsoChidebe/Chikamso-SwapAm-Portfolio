1. Project Overview

Mission Statement

SwapAm revolutionizes campus sustainability by creating a comprehensive platform where university students can exchange, sell, and donate items within their campus community. Our AI-powered system reduces waste, builds connections, and promotes environmental consciousness through smart technology and intuitive design.

Project Scope

SwapAm consists of multiple integrated components:

  • Landing Page: Professional marketing website built with React and Vite
  • Frontend Application: Full-featured React TypeScript application with Material-UI
  • Backend API: Java Spring Boot REST API with PostgreSQL database
  • AI Integration: Smart matching and valuation algorithms
  • Demo System: Standalone HTML/CSS/JS demo for testing

Live Deployment

Production URL: https://swap-am-kappa.vercel.app

The application is deployed on Vercel with full CI/CD integration and automatic deployments from the main branch.
