# Chikamso - SwapAm Portfolio

A modern, responsive portfolio website showcasing the SwapAm project and frontend development skills.

## Features

- **Responsive Design** - Works perfectly on all devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Interactive Elements** - Animated skill bars, image gallery, smooth scrolling
- **Contact Form** - Functional contact form with validation
- **SEO Optimized** - Proper meta tags and semantic HTML

## Sections

1. **Hero Section** - Introduction with call-to-action buttons
2. **About** - Personal information and statistics
3. **Featured Project** - Detailed showcase of SwapAm project
4. **Skills** - Technical skills with animated progress bars
5. **Contact** - Contact information and form
6. **Footer** - Social links and copyright

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Font Awesome Icons
- Responsive Design

## Setup Instructions

1. Open `index.html` in your web browser
2. No additional setup required - it's a static website
3. For development, use a local server for best experience

## Customization

### Adding Images
- Add your profile photo to replace the placeholder in the hero section
- Add SwapAm project screenshots to the `assets/images/` folder
- Update image paths in the HTML

### Updating Content
- Modify personal information in `index.html`
- Update contact details
- Add or modify project details
- Customize color scheme in `style.css`

### Contact Form
The contact form currently shows an alert. To make it functional:
- Integrate with a backend service (Node.js, PHP, etc.)
- Use a service like Formspree, Netlify Forms, or EmailJS
- Update the form action and method attributes
